# フロントエンド開発ガイド（front/household-expenses）

Vue.js によるフロントエンド開発時に参照する。

## 開発コマンド

```bash
cd front/household-expenses
npm run dev             # 開発サーバー起動
npm run build           # プロダクションビルド
npm run start           # プロダクションサーバー起動
npm run lint            # ESLint実行
```
